# 🎬 MovieHub

Sistema em Java para busca de filmes utilizando a API OMDb.

## 🚀 Funcionalidades
- 🔍 Busca de filmes por nome
- 🌐 Integração com API OMDb
- 📄 Exibição de informações (título, ano, gênero, sinopse)
- ⭐ Adição de filmes aos favoritos
- 🚫 Prevenção de duplicatas nos favoritos

## 🛠 Tecnologias utilizadas
- Java
- HTTP Client (Java 11+)
- Gson
