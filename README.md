# 🎬 MovieHub

Sistema em Java para busca de filmes utilizando a API OMDb.

## 🚀 Funcionalidades
- 🔍 Busca de filmes por nome
- 🌐 Integração com API externa

## 🛠 Tecnologias utilizadas
- Java
- HTTP Client (Java 11+)
